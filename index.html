Comparative Analysis of Various ML Models for Heart Attack Analysis

Group: Minura Samaranayake, Harsh Dubey, Hasan Mirzakhani, Dhe Yeong
Presentation | Report
2023

In this study, we rigorously assessed various machine learning (ML) models to predict heart attack risks using a well-prepared and balanced dataset. Our exploration spanned multiple classifiers, including Logistic Regression, K-Nearest Neighbors, Random Forest, Decision Tree, Gradient Boosting Machine, Naive Bayes, Neural Networks, and Support Vector Machine. We introduced three variants for each classifier: a base model, a 'Random Tuned' model with hyperparameters adjusted randomly, and a 'Grid Tuned' model with hyperparameters fine-tuned methodically. Furthermore, we integrated these variants into a voting classifier ensemble for each model category.
